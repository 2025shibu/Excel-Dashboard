
## 📊 Swiggy Sales Dashboard – Excel Project

This project presents a comprehensive sales dashboard built in **Microsoft Excel**, designed to analyze and visualize food delivery performance data from Swiggy across India.

### 🧩 Project Overview
The dashboard transforms raw transactional data into actionable insights using Excel’s powerful data analysis and visualization tools. It highlights key performance indicators (KPIs) and trends to support business decisions in the food delivery domain.

### 📌 Key Features
- **KPI Metrics**: Total Sales, Average Rating, Order Volume, Ratings Count, and Average Order Value.
- **Trend Analysis**:
  - Monthly and Daily Sales Trends
  - Weekly Performance Overview
  - Quarterly Summary combining Sales, Ratings, and Orders
- **Segmented Insights**:
  - Sales by Food Type (Veg vs Non-Veg)
  - State-wise Sales Distribution (Map Visualization)
  - Top 5 Cities by Revenue Contribution
- **Interactive Visuals**: Bar charts, line graphs, pie charts, and heat maps for intuitive storytelling.

###  Tools & Techniques Used
- **Microsoft Excel**: Core platform for data cleaning, aggregation, and dashboard creation.
- **Pivot Tables & Charts**: For dynamic data summarization and visualization.
- **Conditional Formatting**: To highlight performance patterns.
- **Map Charts**: For geographic sales distribution.
